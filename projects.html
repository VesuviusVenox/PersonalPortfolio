Minecraft Indev+

Indev+ is a complete overhaul mod for Minecraft Indev, which aims to take an old 2010 version of the game and shape it into something entirely new and unique.

The mod contains a plethora of new features and mechanics, which help add a consistent progression system into the game, whilst simultaneously keeping the finite worlds philosophy of the original.

Some unique elements of the mod include, but are not limited to: Seasons, complex character system, world hopping using portals, end-game items (charms), blood moon mechanic, new world types, boss mobs and much more.

Minecraft Classic+

Classic+ is a complete overhaul mod for Minecraft Classic version 0.30, which adds a plethora of new, original content to the game, whilst keeping the look and feel of the original version.

Some of the more notable additions include, but are not limited to: World types and themes, new "arcade" gamemode, powerups and unique crafting systems, single-player commands, survival inventory, armor and much more.

The mod also supports a fully functioning survival multiplayer, something the original game was lacking. This was made by repurposing the little existing MP code and writing a new netcode specifically designed to support the survival elements of the game.

This project was what got me into Java programming, as my only prior experience has been minor code edits and making small programs using Processing. About 55% of the game's original code has been rewritten, and a lot of new code has been added.

Nitronic Rush Custom Level - CORE Filtration

Some of my earliest experience with level design, CORE Filtration is the second level I made for the free survival racing game "Nitronic Rush". It was made using the game's built-in level editor in the span of several days, and it is more well received than my previous level, often denoted as the hardest level ever made for the game.

To the left is a full run of the level from a speedrunner of the game. It is 11 floors, each with obstacles of progressively increasing difficulty. The level is a "challenge" level, meaning if you die, you get sent back to the very beginning.

Notable takeaways:

  • First experience with level design in any game.
  • Positive feedback from the community.
  • Unique, never tried in the game before concept.

RubyRevived - Minecraft Pre-Classic mod

RubyRevived is a mod for Minecraft Pre-Classic which I briefly collaborated on. The mod aims at recreating the lost game "RubyDung", which served as some of Minecraft's inspiration and early code-base. I helped with getting block color offsets working, as well as fixing some menu and world loading bugs.

The project is currently on hiatus, but I am still a member of the development team.
