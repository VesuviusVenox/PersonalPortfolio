Bio

Minecraft Recreation Project

A project aimed at recreating every individual missing version of Minecraft with extreme accuracy.

Over the span of two years (from September 2019 until June 2021) I have managed to successfully recreate 151 missing versions of the game Minecraft, all coded in Java using LWJGL and by modifying the source of existing versions.

This project was my introduction to Java, and helped me improve my programming skills along the way.

To the is an image of one of 10 builds labeled "0.24", all of which took 2 days to make.

A full spreadsheet of all the different recreations and an overview of how each one was made can be found here.

Notable takeaways:

  • Learned the basics and Java and LWJGL by modifying existing game code.
  • Learned the basics of deobfuscating code by hand.
  • Learned how to use the Eclipse IDE and an array of bytecode editors.
  • Lots of dedication and research.

Pokemon Crimson

My first attempt at a video game. The project began in early concepting stages as early as 2016, where I wanted to make my own Pokemon video game with an aging and maturing protagonist, set in a darker, more realistic version of the Kanto region.

I also had game mechanic ideas, such as a new type: Light, as well as a bunch of new moves and other gameplay elements.

To the left is a short video demo of an older version of the project, showing off the starter house. The current version has the entirety of Pallet Town finished, the new light type and some new moves and a sprite for the main character. I went for high amounts of detail with the level design.

The game is made in RPG Maker using Pokemon Assets from the third generation of games for the GameBoy Advance.

Notable takeaways:

  • Learned how to use PRG Maker and basic scripting in Ruby.
  • Learned how to use GIMP for sprite editing.
  • Entry point for my study at Saxion; was tasked to create a simple game using any tools of my liking. Ended up showing the later version of the demo during the intake.

Bio

Nitronic Rush Custom Level - CORE Filtration

Some of my earliest experience with level design, CORE Filtration is the second level I made for the free survival racing game "Nitronic Rush". It was made using the game's built-in level editor in the span of several days, and it is more well received than my previous level, often denoted as the hardest level ever made for the game.

To the left is a full run of the level from a speedrunner of the game. It is 11 floors, each with obstacles of progressively increasing difficulty. The level is a "challenge" level, meaning if you die, you get sent back to the very beginning.

Notable takeaways:

  • First experience with level design in any game.
  • Positive feedback from the community.
  • Unique, never tried in the game before concept.

RubyRevived - Minecraft Pre-Classic mod

RubyRevived is a mod for Minecraft Pre-Classic which I briefly collaborated on. The mod aims at recreating the lost game "RubyDung", which served as some of Minecraft's inspiration and early code-base. I helped with getting block color offsets working, as well as fixing some menu and world loading bugs.

The project is currently on hiatus, but I am still a member of the development team.

Discord server for the project here

Notable takeaways:

  • Team collaboration.
  • Improved Java knowledge.
  • Positive community feedback.
